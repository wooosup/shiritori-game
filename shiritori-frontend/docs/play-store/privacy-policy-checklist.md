# Privacy Policy Checklist (Android)

Google Play 제출 전 개인정보처리방침 점검표입니다.

## 1) 문서 기본

1. 접근 가능한 URL이 존재한다
2. 문서 언어가 사용자 대상(한국어)에 맞다
3. 최종 개정일이 표기되어 있다
4. 운영자 문의 이메일이 실제 값으로 설정되어 있다

## 2) 필수 기재 항목

1. 수집 항목(이메일/프로필/게임 데이터 등)
2. 이용 목적(인증, 게임 기능, 랭킹, 안정성)
3. 제3자 처리(인증/호스팅 인프라)
4. 보관/삭제 기준
5. 이용자 권리(정정/삭제/탈퇴)
6. 보안 조치(전송 암호화, 접근 통제)

## 3) Play Console 연결 체크

1. Play Console `App content > Privacy policy`에 URL 등록
2. 앱 내부에서도 정책 접근 경로 제공(옵션/설정 화면 링크 권장)
3. Data safety 답변과 정책 문구가 모순되지 않음

## 4) 이 프로젝트 기준 파일

1. 정책 페이지: `public/legal/privacy-ko.html`
2. 계정 삭제 안내: `public/legal/account-deletion-ko.html`
3. Data safety 기준표: `docs/play-store/data-safety-matrix.md`

## 5) 배포 전 마지막 점검

1. 정책 링크가 200 OK로 열리는지 확인
2. 모바일(WebView)에서도 렌더링 문제 없는지 확인
3. 계정 탈퇴 실행 후 실제 데이터 삭제가 되는지 확인
